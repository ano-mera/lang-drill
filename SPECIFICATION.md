# TOEIC Part7 文書問題アプリ 仕様書

## 1. アプリ概要

### 目的

- TOEIC Part7（読解問題）の練習に特化した学習アプリ
- 実際の TOEIC 試験に近い形式で問題を出題
- 単一の文書問題に焦点を当てた学習体験
- AI 翻訳機能による学習支援
- **図表付き問題の対応**

### 対象ユーザー

- TOEIC 受験者
- 英語読解力を向上させたい学習者
- 日本語翻訳を活用したい学習者

## 2. 主要機能

### 2.1 文書表示機能

- TOEIC Part7 形式の文書（メール、手紙、広告、記事など）を表示
- 読みやすいレイアウトとフォントサイズ
- 文書の種類に応じた適切なスタイリング
- **TOEIC 形式の指示文表示**：「Questions 1-3 refer to the following email.」
- **本文と問題文の区切り線**
- **図表の表示機能**：本文の下に TOEIC 形式の図表を表示

### 2.2 問題出題機能

- 文書に関する複数の選択肢問題
- 実際の TOEIC 形式（4 択問題）
- **選択肢形式：(A) (B) (C) (D)**
- 問題文と選択肢の明確な表示

### 2.3 解答・採点機能

- ユーザーの選択を記録
- 即座の正誤判定
- 正解の表示
- 解説の提供

### 2.4 AI 翻訳機能

- **OpenAI GPT-4o による自動翻訳生成**
- **本文、問題文、選択肢の日本語翻訳**
- **翻訳ボタンによる表示/非表示切り替え**
- **翻訳がない場合の適切な表示**

### 2.5 進捗管理

- 解答状況の記録
- 正答率の表示
- 新しい問題のランダム選択

### 2.6 図表表示機能

- **TOEIC 公式問題集形式の図表表示**
- **モノクロ表現**：色や装飾は一切使用しない
- **テーブル形式**：シンプルな表形式での表示
- **レスポンシブ対応**：モバイルでも適切に表示

## 3. データ構造

### 3.1 JSON データ形式

```json
{
  "id": "unique_id",
  "title": "文書タイトル",
  "type": "email|letter|advertisement|article",
  "content": "文書の本文",
  "questions": [
    {
      "id": "question_id",
      "question": "問題文",
      "options": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
      "correct": "A",
      "explanation": "解説文"
    }
  ],
  "chart": {
    "type": "table",
    "title": "図表タイトル",
    "description": "図表の説明（オプション）",
    "data": [
      {
        "Column1": "Value1",
        "Column2": "Value2"
      }
    ]
  },
  "hasChart": true,
  "metadata": {
    "difficulty": "easy|medium|hard",
    "estimatedTime": 300,
    "wordCount": 150,
    "questionCount": 3,
    "passageType": "email",
    "topic": "business"
  },
  "contentTranslation": "本文の日本語翻訳",
  "questionTranslations": ["問題文1の翻訳", "問題文2の翻訳"],
  "optionTranslations": [
    ["選択肢A1", "選択肢B1", "選択肢C1", "選択肢D1"],
    ["選択肢A2", "選択肢B2", "選択肢C2", "選択肢D2"]
  ]
}
```

## 4. UI/UX 設計

### 4.1 レイアウト

- レスポンシブデザイン
- 文書表示エリア（上部）
- **図表表示エリア（本文の下）**
- 区切り線
- 問題表示エリア（下部）
- 翻訳表示エリア（答え合わせ時のみ）

### 4.2 デザイン原則

- TOEIC 試験に近い見た目
- 読みやすさを重視
- シンプルで直感的な操作
- **白背景、シンプルなデザイン**
- **本文と問題文の横幅統一**
- **図表はモノクロ、枠線のみ使用**

### 4.3 翻訳表示

- **答え合わせ画面でのみ翻訳ボタン表示**
- **本文翻訳、問題文翻訳、選択肢翻訳の個別表示**
- **翻訳がない場合の適切なメッセージ表示**

### 4.4 図表表示

- **TOEIC 公式問題集形式**：シンプルな表形式
- **モノクロ表現**：背景色、色付きフォントは一切使用しない
- **枠線と文字強調**：太字、大文字、記号による強調のみ
- **レスポンシブ対応**：モバイルでも適切に表示
